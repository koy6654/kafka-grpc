## Skills
NextJS, Kafka, gRPC, Docker

## Description
NextJS를 사용한 Kafka 모듈이며 내부에서 사용하기 위해 gRPC를 추가함

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
